# MoodSync: CLI Mood Music Recommendation System

## Description
MoodSync is a Python CLI application that helps users organize and manage songs based on moods, genres, and listening preferences. The system allows users to search, sort, analyze, and receive mood-based music recommendations.

This project demonstrates Object-Oriented Programming, file handling, data structures, algorithms, and advanced Python concepts.

---

# Features
- Add and remove songs
- Search songs by title, artist, genre, or mood
- Mood-based song recommendations
- Sort songs by different categories
- Favorite songs system
- Play count tracking
- Music statistics
- JSON data saving and loading
- Error and duplicate handling and input validation

---

# Technologies Used
- Python 3
- JSON File Handling
- Object-Oriented Programming

## How to Run the program:
Go to the folder you downloaded or extracted and open it in the terminal and run the following command:
#### For Windows:

#### For Linux/Mac:
